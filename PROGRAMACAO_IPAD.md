# Programação no iPad: Linguagens e Aplicativos

Embora o iPad não seja um ambiente de desenvolvimento tradicional como um PC ou Mac, ele se tornou uma ferramenta poderosa para programação graças a diversos aplicativos especializados. Abaixo está uma visão geral das linguagens de programação que você pode usar no iPad e os aplicativos específicos recomendados para cada uma.

## 1. Swift
A linguagem nativa da Apple, ideal para criar aplicativos para iOS e iPadOS.
* **Swift Playgrounds:** Aplicativo oficial da Apple. Começou como uma ferramenta de aprendizado, mas hoje permite criar aplicativos completos (usando SwiftUI) e até publicá-los diretamente na App Store pelo próprio iPad.

## 2. Python
Uma das linguagens mais populares, excelente para automação, scripts rápidos e ciência de dados.
* **Pyto:** Um ambiente Python muito completo (inclusive mencionado na arquitetura de scripts deste repositório). Tem excelente suporte a bibliotecas puras do Python, acesso a recursos do sistema iOS e integração com o app Atalhos.
* **Pythonista:** Uma ferramenta clássica e muito poderosa para Python no iOS. Permite criar interfaces gráficas personalizadas e acessar APIs nativas do dispositivo (como câmera e contatos).
* **Carnets e Juno:** Aplicativos focados em rodar Jupyter Notebooks localmente no iPad, ideais para análise de dados usando pandas e matplotlib.

## 3. JavaScript, HTML e CSS (Web & Automação)
A base da web e de muitas ferramentas de automação modernas.
* **Scriptable:** Um aplicativo fantástico que permite escrever scripts em JavaScript para integrar-se ao iOS. É muito usado para criar Widgets personalizados para a tela inicial (também suportado por este repositório para integração de APIs).
* **Play.js:** Um ambiente focado em Node.js e React Native. Permite rodar e visualizar projetos web complexos e aplicativos diretamente no iPad.
* **Textastic e Koder:** Editores de código avançados (como o VS Code, mas mais leves). Eles oferecem *syntax highlighting* para dezenas de linguagens e são perfeitos para desenvolvimento web estático.

## 4. Lua
Uma linguagem leve e rápida, frequentemente usada para scripting e desenvolvimento de jogos.
* **Codea:** Um aplicativo incrível e visual para criar jogos 2D/3D e simulações complexas usando Lua. Ele oferece acesso direto aos sensores do iPad (acelerômetro, toque) e um ambiente de renderização poderoso.

## 5. C e C++
Para desenvolvimento de sistemas, algoritmos ou aprendizado escolar.
* **Mobile C:** Um compilador C e C++ offline que permite compilar e rodar códigos localmente no iPad. Útil para estudantes e testes rápidos de algoritmos.

## 6. Desenvolvimento Remoto (Qualquer Linguagem)
Se você precisa usar **Java, C#, Go, Rust, Ruby**, ou frameworks pesados (como Docker ou bancos de dados completos), a abordagem recomendada no iPad é a nuvem ou conexão remota.
* **GitHub Codespaces / Gitpod:** Rodando diretamente no navegador Safari, você pode ter um ambiente VS Code completo (com terminal Linux) hospedado na nuvem.
* **Blink Shell e Termius:** Os melhores clientes SSH/Mosh para iPad. Você pode se conectar a um servidor remoto, um Raspberry Pi ou seu próprio PC e desenvolver usando o terminal (`vim`, `nano`, etc.).

## 7. Ferramentas Essenciais Adicionais
Para completar seu fluxo de trabalho no iPad, estes apps são fundamentais:
* **Working Copy:** O melhor cliente Git para iOS/iPadOS. Ele permite clonar repositórios (como o GitHub), fazer commits, pushes e expor os arquivos para outros aplicativos editarem.
* **Inspect Browser:** Como o Safari no iPad não tem "Inspecionar Elemento", este aplicativo adiciona as famosas ferramentas de desenvolvedor (DevTools) para inspecionar HTML/CSS e debugar JavaScript no dispositivo.
